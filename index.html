Current project

LexiGrid is the first game released by XoRdbX Studio. It is a word puzzle game built around a dynamic grid where players swap letters to form words under time pressure.

LexiGrid

LexiGrid – Letter Swap Quest is a fast-paced word puzzle game for Android. Players swap letters on a 10×10 grid to create words before time runs out. Each stage introduces new layouts and challenges, encouraging quick thinking and strategic choices. Obstacles such as slime and ice tiles add an extra layer of puzzle solving, while each successful word rewards players with additional time to keep the game going. Designed to stimulate vocabulary and cognitive skills, LexiGrid combines learning and entertainment in a visually clean and accessible experience.
